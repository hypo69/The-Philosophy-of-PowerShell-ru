–ü—Ä–µ–∂–¥–µ —á–µ–º –≤—ã–ø–æ–ª–Ω—è—Ç—å –Ω–µ–æ–±—Ä–∞—Ç–∏–º–æ–µ –¥–µ–π—Å—Ç–≤–∏–µ, —Ç–∞–∫–æ–µ –∫–∞–∫ —É–¥–∞–ª–µ–Ω–∏–µ –º–µ—Ç–∞–¥–∞–Ω–Ω—ã—Ö, –ª—É—á—à–µ –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–∏—Ç—å –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—é –≤—ã–±–æ—Ä.

### –ü—Ä–∏–º–µ—Ä 5: –ö–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ –º–µ—Ç–∞–¥–∞–Ω–Ω—ã—Ö —Å –æ–¥–Ω–æ–≥–æ —Ñ–∞–π–ª–∞ –Ω–∞ –¥—Ä—É–≥–∏–µ

–ü—Ä–µ–¥—Å—Ç–∞–≤—å—Ç–µ, —É –≤–∞—Å –µ—Å—Ç—å "—ç—Ç–∞–ª–æ–Ω–Ω—ã–π" —Ñ–∞–π–ª —Å –ø—Ä–∞–≤–∏–ª—å–Ω—ã–º–∏ –º–µ—Ç–∞–¥–∞–Ω–Ω—ã–º–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –∞–≤—Ç–æ—Ä, –∫–æ–ø–∏—Ä–∞–π—Ç), –∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ —Å–∫–æ–ø–∏—Ä–æ–≤–∞—Ç—å –∏—Ö –Ω–∞ –¥—Ä—É–≥–∏–µ —Ñ–æ—Ç–æ.

```powershell
# 1. –í—ã–±–∏—Ä–∞–µ–º —ç—Ç–∞–ª–æ–Ω–Ω—ã–π —Ñ–∞–π–ª
$sourceFile = Get-ChildItem "D:\Photos" -Filter "*.jpg" | Out-ConsoleGridView -Title "–í—ã–±–µ—Ä–∏—Ç–µ –≠–¢–ê–õ–û–ù–ù–´–ô —Ñ–∞–π–ª"

    if ($targetFiles) {
        & exiftool.exe -TagsFromFile $sourceFile.FullName ($targetFiles.FullName)

    }
}
```
